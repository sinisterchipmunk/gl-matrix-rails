# gl-matrix-rails

A gem to automate using [gl-matrix](http://glmatrix.net/) with Rails 3.

## Installation

Add this line to your application's Gemfile:

    gem 'gl-matrix-rails'

And then execute:

    $ bundle

## Usage

Just add the following to your Rails asset manifest in
`app/assets/javascripts/application.js`:

    //= require gl-matrix
